# Tower of Hanoi dengan Stack (C++)

Tugas praktikum Bab 6 – Stack.  
Program ini mensimulasikan pemindahan 3 batu (50, 75, 100) dari tiang A ke tiang B dengan bantuan tiang C, menggunakan prinsip **LIFO (Last In First Out)** tanpa aturan ukuran batu (sesuai permintaan soal).

## Langkah yang dijalankan (manual oleh user)
1. `A C` → pindah 50 dari A ke C
2. `A C` → pindah 75 dari A ke C
3. `A B` → pindah 100 dari A ke B
4. `C B` → pindah 75 dari C ke B
5. `C B` → pindah 50 dari C ke B
